**not for production**

# Development
Install node 8
Install deps
```sh
npm i
```


Run backend

```sh
go build && ./chantra
```

Run frontend
```sh
npm run dev
```

# Tests and lint

Run tests once
```sh
npm test
```

Run watch mode
```sh
npm run test-watch
```

Run lint and fix errors
```sh
npm run lint
```
